
SETTING UP THE PROJECT

1.Real Android devices are used when running the tests.
2.Java 1.8 or above. 
3.Gauge. Gauge works with Java 1.6 and above.
4.Gauge Plugins: Java plugin, HTML Report plugin, XML Report plugin
5.Appium
6.Set up Maven
7.adb


RUN THE PROJECT

1.Set the device UDID in src -> test -> Java -> HookImpl class
2.Set app package and activity in src -> test -> Java -> HookImpl class
3.Connect the android device with usb cable
4.Run the specs
